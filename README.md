### Usage
- Compile it.
- Call `WMDict` with the search word as argument.


(You can create alias in `~/.bash_profile` like `alias dict="~/somePath/WMDict"`)

#### or 

Add this shebang at the top of `WMDict.swift`

`#!/usr/bin/env xcrun swift`

then

`chmod +x WMDict.swift`
